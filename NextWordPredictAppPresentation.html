Predict Next Word App

Scot Shields
8/17/2015

ShinyApp Description

  • To create a ShinyApp which predicts the next word in an incomplete sentence.

  • The app will prompt the user for an incomplete sentence or phrase, then suggest the most likely next word in the incomplete sentence based on analysis of text samples.

  • The samples are drawn from news, blogs, and twitter.

Description of Algorithm

Preprocessing

  1. Randomly sampled lines of english text from blogs, news feeds, and twitter to create corpus
  2. Cleaned text sample using tm_map function from tm package
  3. Used content_transformer to replace “/|@|\|” with space and to convert text to lowercase
  4. Removed numbers, punctuation, and whitespace

Description of Algorithm (continued)

  1. Tokenized text using function from Rweka package
  2. Created Term document matrix of tokenized text
  3. Created table from term document matrix with final word, preceeding words without final word, and probability of final word following from preceeding words in ngram
  4. Matching function then takes input sentence fragment and finds largest matching ngram in table with highest probability final word, and returns final word.

Example: Given input “a chance” algorithm predicts “to”

ShinyApp Instructions

  1. Enter incomplete sentence
  2. Press the submit button

  3. Predicted next word appears to the right
